4. Diagnóstico — Cascata de Erros
A causa-raiz é a superestimação da fração argila pelo SoilGrids nesta região do Grupo Caiuá (arenitos, MS). A argila real é provavelmente caulinítica (mineral 1:1, inerte), mas as PTFs assumem argila ativa.
Erro Primário: SoilGrids estima 31.3% argila — Solo real é essencialmente arenoso (NP, A-2-4)

Problemas reais não flagados
Por que o Risk Score é 0 apesar dos problemas reais?
Os 8 risk flags testam limiares que não são atingidos com os dados SoilGrids: Ksat < 1.5 → não (3.72)  |  K > 0.35 → não (0.14)  |  PI > 25 → não (14.6)  |  Shrink ≥ Mod-High → não (Moderate)  |  Compaction ≥ 0.9 → não (0.80)  |  CBR < 5 → não (12.9)  |  Impediment → não  |  USCS problem → não (SC)
Os problemas reais (solo fofo SPT 1-4, colapsividade, alta permeabilidade) são intrinsecamente não-detectáveis por satélite.
5. Parâmetros Não Capturados pela API
Dados que o laudo de campo fornece e que a API não consegue derivar a partir de sensoriamento remoto.
SPT / Resistência à Penetração Requer sondagem mecânica in situ. Essencial para projeto de fundações.
Colapsividade Ensaio de adensamento edométrico com inundação. Crítico para solos porosos.
Recalques Calculado via oedométrico ou simulação numérica (Plaxis 3D). Exigência da apólice.
Agressividade ao Concreto Ensaio químico em laboratório. Requisito do seguro de engenharia.
Nível do Lençol Freático Identificado nas sondagens SPT. Impacta fundações e drenagem.
6. Veredito

Screening Válido com Ressalvas Significativas

A API acerta a decisão de alto nível (fundação rasa viável) mas erra nos parâmetros quantitativos para solos arenosos do Grupo Caiuá. O produto funciona como triagem inicial para priorizar investigações de campo, mas não substitui sondagens SPT e ensaios laboratoriais conforme NBR 6122:2019 e NBR 8036:1983.

Convergências
  • Fundação rasa / radier viável — decisão go/no-go correta
  • Risco de erosão desprezível — ambas as fontes concordam
  • Homogeneidade do perfil — variação <3% de argila
  • Tendência arenosa identificada (>57% areia)
Divergências Críticas
  • Plasticidade: IP=14.6 estimado vs NP real — falso positivo
  • Permeabilidade: erro de 5 ordens de grandeza (3.72 vs 1.224.000 mm/h)
  • Textura: 31% argila estimada em solo essencialmente arenoso
  • CBR subestimado em 33-43% (12.9 vs 19-23%)
  • Risk Score 0 onde há solo fofo e colapsividade real
